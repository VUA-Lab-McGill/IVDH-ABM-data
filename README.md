# IVDH-ABM-data
Simulation output data for empirical and extrapolated hydrogel conditions in the IVDH-ABM.

- 1kPa_H.xlsx: simulation output for the 1kPa-high molecular weight alginate hydrogel condition
- 1kPa_L.xlsx: simulation output for the 1kPa-low molecular weight alginate hydrogel condition
- 3kPa_H.xlsx: simulation output for the 3kPa-high molecular weight alginate hydrogel condition
- 3kPa_L.xlsx: simulation output for the 3kPa-low molecular weight alginate hydrogel condition
- In_Silico.xlsx: simulation output for the optimized alginate hydrogel formulation (200 kDA MW alginate at 1.95% w/v, 0.02 M calcium crosslinker) with E = 3.66 kPa.
